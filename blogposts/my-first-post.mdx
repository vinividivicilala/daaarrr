There are always three things to consider when it comes to animating props:

1. The `type` of element
2. The `children` for the element
3. The `animationDuration` that you prefer

They can be difficult to balance at times, but when each of them are orchestrated in harmony -- the animation will be sublime.

## Deciding our approach

```tsx {9}
// components/CheckoutCard.tsx

import CheckoutItem from "components/CheckoutItem";
import useItems from "hooks/useItems";

interface CheckoutCartProps {
  id: number;
}

export default function CheckoutCart({ id }: CheckoutCartProps) {
  const items = useItems(id);

  return (
    <div>
      {items.map((item) => (
        <CheckoutItem key={item.id} item={item} />
      ))}
    </div>
  );
}
```
